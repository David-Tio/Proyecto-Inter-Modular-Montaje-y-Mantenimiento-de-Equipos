## Información General

Este repositorio contiene el trabajo correspondiente al módulo **Montaje y Mantenimiento de Equipos** del Proyecto Intermodular.

La empresa simulada para este proyecto es **Chamartín**, una oficina pequeña con dos plantas y un máximo de 20 empleados.

- **Presentación:** `Chamartin_Montaje_Mantenimiento.pdf`

## Contenido del Proyecto

El trabajo cumple con todos los apartados exigidos en el módulo:

1. **Introducción**
2. **Análisis de necesidades de Hardware**  
   - Tipos de dispositivos necesarios  
   - Justificación de 20 ordenadores, 2 impresoras, 1 servidor, equipos más potentes, etc.
3. **Configuración de los equipos**  
   - Configuración para puestos de **Administración y RRHH** (Intel Core i5, 16 GB RAM, SSD 512 GB…)  
   - Configuración para puestos de **Producción y Ciberseguridad** (Intel Core i7/i9, mayor RAM…)
4. **Montaje del equipo**
   - Montaje realizado mediante los pasos explicados en clase: placa base en una superficie no conductora, montamos la ram en la placa base, la cpu, etc.
   - Despues del montaje realizar el primer encendido para ver si los componentes estan en un estado correcto.
   - Si esta todo correcto lo siguiente seria instalar el sistema operativo mediante un usb booteable.
6. **Mantenimiento básico** *(pendiente de ampliar)*
7. **Inventario de equipos** *(pendiente de ampliar)*

## Objetivo del Módulo

Demostrar la capacidad de:
- Identificar las necesidades reales de hardware de la empresa **Chamartín**.
- El realizar la seleccion de los componentes adecuados y equilibrados según el uso de cada puesto.
- Para entender el proceso de montaje de un equipo informático.
- Tambien para poder explicar las tareas básicas de mantenimiento para alargar la vida útil de los equipos.
